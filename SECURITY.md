# Security Policy

## Supported Versions
Currently, only the latest version of the **Ultimate Tracker Aggregator** on the `main` branch is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

As this is a God-Tier automation script maintained by **mrgusux**, security and data integrity are our highest priorities.

If you discover a security vulnerability, an injection flaw in the sanitization pipeline, or any bypass mechanism, please **DO NOT** open a public issue. 

Instead, please report it privately to **mrgusux** via GitHub direct messaging or email (if provided on the GitHub profile). We will review the vulnerability and patch it within 48 hours.
