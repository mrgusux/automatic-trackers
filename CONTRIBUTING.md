# Contributing to Ultimate Tracker Aggregator Pro Max

First off, thank you for considering contributing to this God-Tier repository! 🎉
This project is maintained and architected by **mrgusux**.

## How Can You Contribute?

### 1. Adding New Tracker Sources
If you have found a highly reliable, public, and open-source tracker list that is not currently in our script:
* Do **NOT** remove any existing URLs.
* Open a **Pull Request (PR)** adding the new URL to the `SOURCES` array in the `update-trackers.yml` file.
* Ensure the source is reliable (uptime > 95%).

### 2. Improving the Sanitization Pipeline
If you know advanced `awk`, `sed`, or `grep` commands to filter out fake IPs or bad payloads even better, feel free to submit a PR!

## Pull Request Process
1. Fork the repo.
2. Create a new branch (`feature/new-trackers`).
3. Commit your changes.
4. Push to the branch and open a Pull Request against our `main` branch.
5. Wait for **mrgusux** to review and merge your code.
