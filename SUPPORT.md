# Support Guidelines

Need help with the **Ultimate Torrent Tracker Aggregator Pro Max**? Here is where you can find answers and ask questions:

## 💬 GitHub Discussions
For general questions, how to use this with your torrent client (qBittorrent, Aria2, etc.), or to share ideas, please use the GitHub Discussions tab.

## 🐛 Bug Reports
If you found a reproducible bug in the script or pipeline, please open an Issue using our structured Bug Report template.

## 🚀 Feature Requests
Got an idea for a new tracker source or an advanced feature? Open an Issue using the Feature Request template.

*Note: Please do not open issues for general tech support or non-bug-related questions. Help keep our issue tracker clean for development work.*
